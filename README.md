## Project Overview
This project analyzes historical stock price data and forecasts future stock prices using machine learning techniques.

It also compares multiple stocks based on:
- Average Returns
- Risk (Volatility)
- Risk vs Return relationship

## Stocks Analyzed
- ICICI Bank
- Reliance Industries
- Infosys
- HDFC Bank
- Tata Consultancy Services

## Features
- Historical stock data collection
- Moving average calculation
- Daily returns analysis
- Future 7-day stock forecasting
- Multi-stock comparison
- Risk vs Return visualization

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- yFinance
- Scikit-learn

## Key Insights
- ICICI Bank showed the highest return with higher volatility.
- Reliance showed balanced performance.
- TCS showed lower volatility and stable performance.
